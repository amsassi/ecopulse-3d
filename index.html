Comprendre EcoPulse 3D

EcoPulse compare la consommation d'energie d'un territoire avec sa production locale d'energies renouvelables. L'objectif est de voir rapidement les zones sous tension et les territoires qui produisent davantage localement.

Carte

Le fond OpenStreetMap sert de repere geographique. Les regions restent plates pour garder une lecture claire.

Anneaux

L'anneau rouge exterieur represente la consommation. L'anneau vert ou bleu interieur represente la production EnR locale.

Flux

Les pictos indiquent le type de site de production. Les pointilles lumineux partent de ces sites et convergent vers le centre de la region, comme une alimentation locale.

Mode futur

Les curseurs simulent plus de solaire local et de renovation thermique. Les anneaux evoluent pour montrer l'effet possible sur l'autonomie.
