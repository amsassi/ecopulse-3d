Comprendre EcoPulse

EcoPulse compare la consommation d'energie d'un territoire avec sa production locale d'energies renouvelables. L'objectif est de voir rapidement les zones sous tension et les territoires qui produisent davantage localement.

Carte

Le fond OpenStreetMap sert de repere geographique. Les regions restent plates pour garder une lecture claire.

Anneaux

L'anneau rouge exterieur represente la consommation. L'anneau vert ou bleu interieur represente la production locale d'energies renouvelables.

Réseau

Les pictos indiquent le type de site de production. Les liens colorés dessinent un maillage local prolongé par des connexions entre régions voisines.

Mode simulation

Les curseurs simulent plus de solaire local et de renovation thermique. Les anneaux evoluent pour montrer l'effet possible sur l'autonomie.

Données énergie

La consommation et la production proviennent de l'Agence ORE Data Fair. EcoPulse interroge les agrégats régionaux et les principaux sites d'energies renouvelables en service.

Fiabilité

Si l'API ou le réseau ne répond pas, l'interface bascule sur un jeu local de secours pour garder la carte consultable sans présenter ces valeurs comme du temps réel.
